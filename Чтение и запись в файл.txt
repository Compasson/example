Чтобы открыть file для посимвольного чтения,
используется класс FileInputReader;
имя файла задаётся в виде строки (String) или объекта File.
Ускорить процесс чтения помогает буферизация ввода, для этого полученная ссылка передаётся в конструктор класса BufferedReader.
Так как в интерфейсе класса имеется метод readLine(),
 все необходимое для чтения фАйла имеется в вашем распоряжении.
При достижении конца файЛа метод readLine() возвращает ссылку null.
